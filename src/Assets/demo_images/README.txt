# Demo Images for README

Place the following images in this directory to complete the README:

1. `portfolio_preview.png` - Main screenshot of the portfolio homepage
   - Recommended size: 1200x630px (social media preview size)
   - Show the entire homepage with navbar and hero section

2. `projects_preview.png` - Screenshot of the projects section
   - Show several project cards to highlight your work
   - Recommended size: at least 1000px wide

3. `skills_preview.png` - Screenshot of the skills/technologies section
   - Show your tech stack visualization
   - Recommended size: at least 800px wide

4. `responsive_preview.png` - Composite image showing responsive design
   - Show the site on multiple devices (desktop, tablet, mobile)
   - Recommended size: 1200x500px

These images will be automatically displayed in the README.md when you add them to this directory.

## How to capture these images:

1. Run your site locally with `npm start`
2. Use browser dev tools to simulate different device sizes
3. Take screenshots (cmd+shift+4 on Mac, Windows+Shift+S on Windows)
4. Edit/crop images as needed and save them with the exact filenames listed above
5. Place them in this directory
